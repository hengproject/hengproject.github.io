What is Autoencoder

An autoencoder is a type of algorithm with the primary
purpose of learning an “informative” representation of the data that
can be used for different applications by learning to reconstruct a set
of input observations well enough.

https://arxiv.org/abs/2003.05991

There are three main components in the autoencoders: encoder. latent feature representation and a decoder.

The encoder and decoder are simply functions, while with the name latent feature representation, one typically intends a tensor of real numbers

https://arxiv.org/pdf/2201.03898
